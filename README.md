# Self_Driving_Car

A kind of deep reinforcement learning project.

The car learns to aim not to bump into any lines user can draw however user like.

## Buttons

+ clear : clear the window.

+ save : save the weights.

+ load : load the wegihts.

## Requirements

+ numpy
+ matplotlib
+ kivy
+ torch
